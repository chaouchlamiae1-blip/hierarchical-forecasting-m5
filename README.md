# Hierarchical Forecasting — M5 Walmart Dataset

## Projet SCM & AI | Master M1 | FSJESTe Spring 2026

### Problématique
Développer un modèle ML pour prévoir la demande au niveau SKU-magasin,
puis la réconcilier avec les niveaux agrégés (famille, rayon, total).
Comparaison Bottom-Up vs Top-Down vs MinTrace OLS.

### Dataset
M5 Competition Walmart — 30 490 SKUs, 10 magasins, 3 États US
Téléchargeable sur : https://www.kaggle.com/competitions/m5-forecasting-accuracy
